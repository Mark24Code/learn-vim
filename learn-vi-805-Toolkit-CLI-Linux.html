VIM学习笔记 工具箱-Linux命令行环境(Toolkit-CLI-Linux)

通过命令行环境（Command-Line Interface, CLI），我们可以更快速地执行操作，并且更方便地自动化一系列繁琐的任务。比如在多个文件中，批量替换指定的字符串等。下文将以Fedora

本文并不会详细介绍每个工具的安装步骤和使用细节，而是聚焦于各个工具与Vim相互配合。文中推荐的工具，完全基于个人主观的偏好，您可以在AlternativeTo网站中寻找功能类似的软件。

Terminal

使用传统的GNOME Terminal，或者可以一键打开的下拉式终端Drop-down Terminal

Zsh

Zsh Bash，除了具有Bash的基本功能之外，还具有更多高级功能：比如交互式制表符补全、自动文件搜索、支持正则表达式、用于定义命令范围的高级速记符，以及丰富的主题引擎等等。

使用包管理器安装Zsh：

$ sudo dnf install zsh
