  • guicursor () (Default: )

  • guifontset () (Default: )

  • guioptions () (Default: )

  • shellslash (ssl) (Default: off)
    使用正斜杠（Forward slash）作为路径分隔符；

  • errorbells (eb) (Default: off)
    是否发出报错响铃；

  • visualbell (vb) (Default: off)
    设置响铃的行为：鸣叫、屏幕闪烁或什么都不做；

  • showmode (smd) (Default: on)
    是否显示当前所处的模式；

  • showcmd(sc) (Default: on, off for Unix)
    是否显示命令信息；

  • cmdheight (ch) (Default: 1)
    设置命令行高度；

  • more (Default: on)
    是否显示“-- More --”提示信息；

  • report (Default: 2)
    设置报告改变行数的阈值；
