VIM学习笔记 工具箱-Firefox

在编写VIM学习笔记的过程中，我需要在Firefox浏览器中，不断预览并验证手工编写的网页；同时也用于预览Markdown格式的Readme文档。

以下将介绍我常用的几款Firefox扩展。

Firefox Screenshots

使用内置的Firefox Screenshots，可以很方便地截取网页。通过鼠标拖拽，就能够滚动屏幕截取超过当前屏显的较长网页。

Hides scrollbars

窗口中的纵向和横向滚动条，除了占用屏幕空间之外，实在想不出还有什么用处？安装Hides scrollbars之后，将自动隐藏滚动条。当然你也可以点击工具栏上的按钮，来重新显示滚动条。

Scroll Progress New

在隐藏滚动条之后，如果你希望了解当前所处的网页位置，那么可以使用Scroll Progress New扩展，在滚动屏幕时显示相对于完整网页的百分比。

相对于冰冷无趣的滚动条，数字化的标识显然更加清晰直观。而且你还可以自定义百分比的显示效果：屏幕位置、字体大小、字体颜色和透明度等。

Html Validator

Html Validator，是基于Tidy和OpenSP开发的，可以在本地验证网页是否符合W3C标准。

Web Standard，可以在不同的浏览器中展示一致的显示效果。

Web Developer

Web Developer可谓网页开发工具的集大成者了。其功能非常丰富，比如检视网页元素、调试CSS、清理Cookie和调整窗口尺寸等等，可以大大提高网页开发和调试的效率。关于更多功能的介绍，请查看chrispederick.com。

GitLab Markdown Viewer

我不是Markdown的fans，我也不太明白：为什么有了HTML这样完整的标签语言，还需要再有一个实现相同功能的子集。无论如何，为了编写Readme文档，我还是需要利用GitLab Markdown Viewer，来预览Markdown文档。
