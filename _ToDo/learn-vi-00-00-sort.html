假设我们需要对以下文本进行排序。

.KS
.IP "TTY_ARGV" 2n
The command, specified as an argument vector,
that the TTY subwindow executes.
.KE
.KS
.IP "ICON_IMAGE" 2n
Sets or gets the remote image for icon's image.
.KE
.KS
.IP "XV_LABEL" 2n
Specifies a frame's header or an icon's label.
.KE
.KS
.IP "SERVER_SYNC" 2n
Synchronizes with the server once.
Does not set synchronous mode.
.KE

其中，每个条目是被“.KS”和“.KE”包围的；而其定义的每个术语则是以“.IP”开头的。而我们只希望对术语进行排序

使用sort命令， As it turns out, you can treat each text block as a unit by joining the block into one line. Here's the first version

Each glossary entry is found between a .KS and .KE macro. j is the ex command to join a line So, the first command joins every glossary entry into one "line." The second command then sorts the file, producing lines like this:

The lines are now sorted by glossary entry; unfortunately, each line also has macros and text mixed in ( [...] 用于省略表示 Somehow, you need to insert newlines to "un-join" the lines. You can do this by modifying your script: mark the joining points of the text blocks before you join them, and then replace the markers with newlines. Here's the expanded ex script:

1）将换行符替换为“@@”字符：

g/^\.KS/,/^\.KE/-1s/$/@@/

.KS@@
.IP "TTY_ARGV" 2n@@
The command, specified as an argument vector,@@
that the TTY subwindow executes.@@
.KE
.KS@@
.IP "ICON_IMAGE" 2n@@
Sets or gets the remote image for icon's image.@@
.KE
.KS@@
.IP "XV_LABEL" 2n@@
Specifies a frame's header or an icon's label.@@
.KE
.KS@@
.IP "SERVER_SYNC" 2n@@
Synchronizes with the server once.@@
Does not set synchronous mode.@@
.KE

2）以“.KS”和“.KE”作为首尾标记来合并行：

g/^\.KS/,/^\.KE/j

.KS@@ .IP "TTY_ARGV" 2n@@ The ... vector,@@ ... .@@ .KE
.KS@@ .IP "ICON_IMAGE" 2n@@ Sets or gets ... image.@@ .KE
.KS@@ .IP "XV_LABEL" 2n@@ Specifies a ... an icon's label.@@ .KE
.KS@@ .IP "SERVER_SYNC" 2n@@ Synchronizes with ... mode.@@ .KE

3）对文本排序：

%!sort

.KS@@ .IP "ICON_IMAGE" 2n@@ Sets or gets ... image.@@ .KE
.KS@@ .IP "SERVER_SYNC" 2n@@ Synchronizes with ... mode.@@ .KE	
.KS@@ .IP "TTY_ARGV" 2n@@ The ... vector,@@ ... .@@ .KE
.KS@@ .IP "XV_LABEL" 2n@@ Specifies a ... an icon's label.@@ .KE

4）将“@@ ”字符替换为换行符：

%s/@@ /^M/g

.KS
.IP "ICON_IMAGE" 2n
Sets or gets the remote image for icon's image.
.KE
.KS
.IP "SERVER_SYNC" 2n
Synchronizes with the server once.
Does not set synchronous mode.
.KE
.KS
.IP "TTY_ARGV" 2n
The command, specified as an argument vector,
that the TTY subwindow executes.
.KE
.KS
.IP "XV_LABEL" 2n
Specifies a frame's header or an icon's label.
.KE

因为在第2步中，使用j命令合并行时，自动新增了一个空格；所以在第3步中，在替换命令里查找“@@”以及紧跟其后的一个空格

通过以上的全局替换命令和外部排序命令，对文档中的特定术语进行了排序，而且保持格式不变。
