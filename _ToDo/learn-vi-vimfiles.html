使用以下命令，可以查看运行时目录：

:set rtp?

vim通常会搜索以下三个运行时目录：

  • $HOME/vimfiles（windows上是你的用户文件夹根目录下的vimfiles ）
  • $VIM/vimfiles ( vim下的vimfiles)
  • $VIMRUNTIME (vimxx，比如vim72)

这样可以在不同的状况加载不同的插件和设置（比如多人共同使用，每个人可以有自己的vim设置）

使用以下命令，可以修改或增加运行时目录

:set runtimepath+=c:/blahblah/vimfiles

你也可以把插件等文件夹放在一个完全不同的地方。

使用:help rtp和:help dos-locations命令，可以查看更多帮助信息。

vimfiles目录中的设置，会覆盖runtime目录中的设置。例如，vimfiles/plugin/myplug.vim 将优先于 $VIMRUNTIME/plugin/myplug.vim 而被载入。

安装目录

使用以下命令，可以查看$VIM所代表的安装目录：

:echo $VIM

对于Linux操作系统，Vim将被安装在以下目录：

/usr/share/vim

对于Windows操作系统，Vim将被安装在以下目录：

C:\Program Files\vim

对于默认安装目录，您不应该修改以上安装目录中的内容，也不应该在其中存放用户相关的文件。主要包括以下内容：

  • vimXX：带上版本号的程序文件夹，可以让不同版本共存。
  • vimfiles 是用户文件夹，你自己装的插件等东西最好放在这里。

用户目录

使用以下命令，可以查看$HOME所代表的用户目录：

:echo $HOME

  • For Windows：C:\users\username
  • For Linux： ~/.vim

用户目录，在vim版本升级时也不会被覆盖。This contains all of our personal modifications so we don’t mess with the default configuration.目录结构可以与安装目录保持一致

在首次使用Vim之前，可以使用以下命令来创建vimrc配置文件：

$ touch ~/.vimrc

Linux: /home/username/.vimrc Windows: c:\documents and settings\username\_vimrc Whatever you change in the personal vimrc file will overrule any previous setting made in the global vimrc file. This way you can modify the entire configuration without having to ever have access to the global vimrc file.
