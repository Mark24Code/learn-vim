• vimXX：带上版本号的程序文件夹，可以让不同版本共存。
  • vimfiles 是用户文件夹，你自己装的插件等东西最好放在这里。

以上文件夹是运行时文件夹，vim通常会搜索以下三个目录下的运行时文件夹：

  • $HOME/vimfiles（windows上是你的用户文件夹根目录下的vimfiles ）
  • $VIM/vimfiles ( vim下的vimfiles)
  • $VIMRUNTIME (vimxx，比如vim72)
这样可以在不同的状况加载不同的插件和设置（比如多人共同使用，每个人可以有自己的vim设置） 你可以在命令模式下运行 :set rtp? 来查看本机运行时目录设置 你可以通过直接修改 rtp 来增加/修改运行时目录,如 :set runtimepath+=c:/blahblah/vimfiles 这样你也可以把插件等文件夹放在一个完全不同的地方。 :help rtp :help dos-locations

$VIM代表，/usr/share/vim或/usr/local/share/vim

:echo $vim vimfiles目录中的设置，会覆盖runtime目录中的设置。例如，vimfiles/plugin/myplug.vim 将优先于 $VIMRUNTIME/plugin/myplug.vim 而被载入。 在Windows下的 $VIM\vimfilel\ 目录，以及Unix下的 $HOME/.vim/ 目录 are for any customizations that should not be over-written when upgrading to a new version of vim.
