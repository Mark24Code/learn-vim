安装配置

Gist.vim需要安装cURL。如果你没有管理员权限安装软件，那么可以直接将curl.exe复制到C:\Users\usernam\AppData\Local\Programs\Git\bin文件夹内。

如果Gist.vim有如下报错：“没有配置GitHub账户”

You have not configured a Github account. Read ":help gist-vim-setup".

那么需要在vimrc配置文件中，设置github_user变量：

let github_user = 'username'

After that, I restarted the editor and invoked the plugin, which then asked for my GitHub password and created a token in /path/to/gVimPortable/Data/settings/.gist-vim.

列表当前用户所有公开的Gist

:Gist -l

识别文件

使用以下命令，将指定ID的Gist载入到当前缓冲区：

:Gist id
