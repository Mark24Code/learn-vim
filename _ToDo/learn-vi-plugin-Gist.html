Gist.vim

安装配置

首先安装Git for windows，并且将以下目录添加到系统的PATH变量中：

  • C:\Program Files\Git\bin
  • C:\Program Files\Git\cmd

在系统命令行中，使用git --version验证安装是否成功：

在系统命令行中，使用以下命令配置GitHub用户：

  • git config --global user.name "username"
  • git config --global user.email "email"

然后下载cURL，并将curl.exe复制到以下目录：

  • C:\Program Files\Git\bin
  • C:\Users\usernam\AppData\Local\Programs\Git\bin

在命令行中，使用curl --version验证安装是否成功：

grab the script (gist.vim) and copy it to your ~/.vim/plugin directory. You'll also need a Gist account and configure git with your account details. You'll need your username and your Github token. You can find the token under Account Settings -> Account Admin.

如果Gist.vim有如下报错：“没有配置GitHub账户”

You have not configured a Github account. Read ":help gist-vim-setup".

那么需要在vimrc配置文件中，设置github_user变量：

let github_user = 'username'

首次调用插件，将询问你输入GitHub密码， 密码将会被保存到~/.gist-vim：

列表当前用户所有公开的Gist

:Gist -l

识别文件

使用以下命令，将指定ID的Gist载入到当前缓冲区：

:Gist id
