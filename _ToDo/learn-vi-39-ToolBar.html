启用/禁用

工具栏（ToolBar）是使用图标来调用菜单的图形界面元素。Vim默认设置是显示工具栏的。我们可以使用以下命令来隐藏工具栏：

:set guioptions-=T

可以通过toolbar选项来控制工具栏的外观：

  • icon  显示工具栏图标
  • text  显示文本
  • tooltips  当光标位于图标上时显示的文字

工具栏按钮

Vim将工具栏视为一种以ToolBar命名的特殊菜单。例如名为ToolBar.New的menu-item是在工具栏上的New图标。Vim编辑器中的每一个基本图标都有两个名字，例如New图标可以用ToolBar.New或是ToolBar.builtin00来表示。你可以使用:help builtin-tools命令，查看关于基本图标的帮助信息。

New,Open,Save,Undo,Redo,Cut,Copy,Paste,Print,Help,Find,SaveAll,SaveSesn,NewSesn,LoadSesn,RunScript,Replace, WinClose,WinMax,WinMin,WinSplit,Shell,FindPrev,FindNext,FindHelp, Make,TagJump,RunCtags. 如果图标并不与基本的图标相匹配,编辑器就会在$VIMRUNTIME/bitmaps目录中进行查找

使用以下命令，可以查看工具栏上的按钮名称：

:aunmenu ToolBar.<Tab>

使用左右方向键，可以选择某个图标项（其中，"|"用于表示分隔符）, 按下Enter将从工具栏上移除此按钮，按下Esc则可以取消选择。

当我们将光标放在图标上时工具栏会显示一个小提示.要定义这个提示,我们可使用下面的命令: :tmenu menu-itme tip 例如下面的命令可以使得当光标放在打开图标上时会显示Open file字样: :tmenu ToolBar.Open Open file

可以使用以下命令，移除工具栏的提示信息：

:tunmenu menu-itme

移除工具栏按钮

You can hide some of the default gvim ToolBar buttons by adding lines like the following to your vimrc:

aunmenu ToolBar.FindNext

新增工具栏按钮

新增工具栏按钮：

an icon=WinMax 1.410 ToolBar.Maximize <C-w>_

其中，1.410表示按钮在工具栏上的位置；icon=Maximize用于指定按钮图标的文件名，如果你只给出了图标的文件名，但没有明确指明文件的路径名，那么Vim将会在$VIMRUNTIME/bitmaps目录进行查找；如果你使用bmp格式的图标文件，那么不需要指明bmp后缀名；最后是点击按钮所执行的命令。

新增工具栏分隔符：

an 1.400 ToolBar.-sep8- <Nop>
