启用/禁用菜单

使用以下命令，可以隐藏菜单栏：

:set guioptions-=m

查看菜单映射

使用以下命令，可以列出所有菜单映射：

:menu

如果只需要显示特定的菜单内容，那么可以使用以下命令：

:menu File

而以下命令，则可以列出指定菜单项的内容：

:menu File.Save

在这个命令的执行结果的每一行的首个字母指明了这个命令的模式.这里的字符是与我们前面提到的mode参数相一致的.

定义新菜单

Vim编辑器所使用的菜单是由文件$VIMRUNTIME/menu.vim定义的.

可以使用以下:menu命令，定义新的菜单内容：

:menu [priority] menu-item command-string

其中，menu-item描述了放置菜单项的路径，比如File.Save表明Save菜单在File菜单下；还可以依次创建子菜单，例如Tabs.Navigation.Next。菜单定义命令示例如下：

:menu 10.340 &File.&Save:w :confirm w

priority，为数字优先级，用于确定放置菜单项的位置。第一个数字10表明在菜单栏上的位置。数字越小越靠近左侧，而数字越大则越靠近右侧；第二个数字340则决定了下拉菜单的位置。数字越小越靠近上部，而数字越大则越靠近下部。

&，用于指定快捷键，例如&File.&Save可以表明用Alt-F键来选择File菜单，而用S键来选择保存菜单。

使用以下命令，将新建Tabs.Next菜单：

:menu Tabs.Next <Esc>:tabnext<cr>

其中，<Esc>用于进入常规模式；<cr>用于执行命令。

我们还可以使用以下指定模式的菜单定义命令：

  • :amenu 正常(Normal),可视(Visual),运算符操作(Operator-pending)
  • :nmenu 正常(Normal)
  • :vmenu 可视(Visual)
  • :omenu 运算符操作(Operator-pending)
  • :imenu 插入(Insert)
  • :cmenu 命令行

:amenu <silent>Tabs.&Previous :tabprevious<cr>

<silent>，将屏蔽命令行的输出。

:amenu Tabs.-SEP- :

-SEP-，可以在下拉菜单中新建用于间隔菜单项的虚线。其必须是以-开头和结尾的唯一名称；同时必须包含命令,b比如用:空命令。

菜单实例

如果希望始终显示自己新建的菜单，那么可以在vimrc文件中增加以下代码。请注意：代码不需要包括冒号（:）。

amenu <silent>Tabs.New :tabnew<cr>
amenu Tabs.-SEPT1- :
amenu <silent>Tabs.&Next :tabnext<cr>
amenu <silent>Tabs.&Previous :tabprevious<cr>
amenu Tabs.-SEPT2- :
amenu Tabs.&Close :confirm tabclose<cr>
