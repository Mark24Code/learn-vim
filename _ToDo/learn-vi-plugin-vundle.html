vundle

GitHub Desktop

配置

set rtp+=$VIM/vimfiles/bundle/Vundle.vim/ call vundle#begin('$VIM/vimfiles/bundle/') 代替首页Quick start中的命令行。

安装插件

vundle支持源码托管在GitHub的插件，你可以在github.com/vim-scripts/上找到vim官网里所有插件的有镜像。

如果希望安装新插件，首先找到其在github.com的网址，然后将其追加至vimrc配置文件中的call vundle#begin()和call vundle#end()之间，最后执行以下命令：

:PluginInstall

卸载插件

要卸载插件，先在 .vimrc 中注释或者删除对应插件配置信息，然后在 vim 中执行 :PluginClean

更新插件

即可删除对应插件。插件更新频率较高，差不多每隔一个月你应该看看哪些插件有推出新版本，批量更新，只需执行 :PluginUpdate

常见问题

如果在使用:VundleInstall命令安装插件时，出现“git不是内部或外部命令”或者“缺少某一个lib”的报错是，那么请将以下目录添加到Path设置中：

C:\Program Files (x86)\Git\libexec\git-core;
C:\Program Files (x86)\Git\bin;

请注意，路径名需要根据你的Git的安装目录作出相应修改。
