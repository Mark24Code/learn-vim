VIM学习笔记 编译源码(Compile Code)-C

启用/禁用

使用以下命令，查看当前gcc的版本：

$ gcc --version

使用以下命令，将gcc设置为编译器：

set makeprg=gcc\ -o\ %<\ %

使用以下命令，将调用'makeprg'选项定义的命令进行编译，并生成文件：

:make

如果编译发现错误，将在QuickFix中显示错误列表，并跳转到第一个错误处；

使用:cw命令，将打开quickfix窗口。使用:cp命令，跳转到上一个错误；使用:cn命令，跳转到下一个错误。

使用以下命令，可以执行编译后的程序：

:!./%:r

set makeprg=gcc\ -o\ %<\ % :make %:r

通过在vimrc配置文件中增加以下自动命令，可以为C语言文件设置编译快捷键：

augroup make_c
	au!
	au FileType c,cpp  set makeprg=gcc\ -o\ %<\ %
	au FileType c,cpp  map <buffer> <leader><space> :w<cr>:make<cr>
augroup end

如果不希望修改'makeprg'选项，那么可以直接定义定义快捷键来完成编译和执行程序的操作：

map <F8> :w <CR> :!gcc % -o %< && ./%< <CR>
