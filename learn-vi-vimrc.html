如果希望保存设置，需要使用以下命令将设置写入文件中：

:mkvimrc file

其中，file是将要写入的文件名。也可以在配置文件中，使用:source file命令包含其它配置文件的内容。

在vim启动过程中，首先查找初始化文件。如果找到就会自动执行文件的内容。而这些初始化文件一般是：


$HOME/.vimrc
$HOME/_vimrc
$HOME/.exrc
$HOME/_exrc

在Unit/Linux系统中则是：

$HOME/.vimrc

在Windows系统中则是：

$HOME/_vimrc
$HOME/.vimrc
$VIM/_vimrc
$VIM/.vimrc
$HOME/_exrc
$HOME/.exrc
$VIM/_exrc
$VIM/.exrc

如果要执行gvim，那么$VIMRUNTIME/menu.vim也会被读进来。

利用:version命令，可以得到配置文件的详细信息。

例如我们可以在.vimrc中写入以下的一些语句：

:syntax on打开语法加亮功能
:set shiftwidth=4设置缩进宽度为4
A single quotation mark (") 备注

修改配置文件后，需要重新启动Vim，或使用:source命令来应用新的设置：

User preferences take precedence over system preferences. This means that if you change an option set in the system vimrc, your preferences will be respected.

:source ~/.vimrc
